# BookMyShow Clone (Frontend UI)

A frontend-only BookMyShow-style interface built with HTML and CSS.  
This repository contains the UI layer and static assets only.

## Project Type

- Frontend/UI project
- No backend code is included in this repository
- Designed to be connected to a separate backend/API service

## Features

- Homepage layout inspired by BookMyShow
- Movie and event sections with poster assets
- Responsive-style visual structure (can be improved further)
- Reusable image assets for banners, cards, and categories

## Tech Stack

- HTML5
- CSS3
- Static image assets

## Folder Structure

```text
bookmyshow - clone/
├── HTML/
│   └── index.html
├── CSS/
│   └── style.css
└── IMAGES/
    └── ...all UI assets
```

## Run Locally

1. Clone or download this repository.
2. Open the project folder.
3. Open `HTML/index.html` in your browser.

Optional (recommended): run with VS Code Live Server for easier development.
